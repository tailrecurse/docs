## Scanning Tokens

Send a token mint address or symbol to scan any token. The bot will display:

- **Token info** and trading options (Swap, Limit Order, DCA)
- **Your holdings** if you own the token
- **Security indicators**:
  - Jupiter verification (verified or not verified)
  - Suspicious flag (flagged or clean)
  - Mint authority disabled (safe or still enabled)
  - Freeze authority revoked (safe or still active)
  - Trade tax (none or percentage)
  - Holder info: total holders, top holders percentage, dev balance percentage, dev migrations
  - LP info: liquidity (with low-liquidity warning), launchpad name, pool creation time
  - Social links
- **Liquidity information**:
  - Token liquidity with warnings for low liquidity tokens
  - Token creation date and time

## Command List

To access commands, tap the menu icon next to the message input or type `/`.

/>

### `/start`

Creates your wallet on first use, or shows the main menu if you already have one.

- **`/start <invite_code>`** -- join via an invite code from an existing user.

### `/menu`

Shows the main menu with your wallet address, balances (top 5 tokens by USD value), and navigation buttons.

### `/swap`

Scan a token to swap. Send a token mint address or symbol, choose buy or sell, and enter an amount.

### `/limit`

Create a limit order at a target price.

- Buy orders execute when the price drops to your target.
- Sell orders execute when the price rises to your target.
- **Expiry**: configurable duration. Presets: 10m, 1h, 1d, 3d, 7d, 30d, or custom. Default is 7 days.

### `/dca`

Set up Dollar Cost Averaging to automate recurring token purchases at set intervals.

### `/holdings`

View your portfolio with PnL tracking. Shows token balances, USD values, unrealized/realized/total PnL per token, and a wallet-level PnL summary. See [Portfolio & Holdings](/trading-bot/holdings).

### `/orders`

View and manage active DCA and limit orders. Cancel orders directly from this screen. See [Active Orders](/trading-bot/orders).

### `/snipe`

Open the Sniper Mode dashboard. Configure automatic buys for new token launches. See [Sniper Mode](/trading-bot/sniper-mode).

### `/copy`

Open the Copy Trader dashboard. Mirror trades from other wallets automatically. See [Copy Trading](/trading-bot/copy-trading).

### `/wallets`

Manage your wallets -- create, rename, archive, delete, import, or export. See [Wallet Management](/trading-bot/wallet).

### `/referral`

View your referral dashboard with earnings, stats, and claim options. See [Referrals & Invites](/trading-bot/referral).

### `/invites`

View and manage your invite codes. Share your invite link with others.

### `/settings`

Open the bot settings menu.

### `/withdraw`

Withdraw tokens to an external wallet.

1. Select a wallet if you have multiple wallets.
2. Enter the recipient Solana address.
