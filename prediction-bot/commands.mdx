To access commands, tap the menu icon (☰) next to the message input or type `/`

### `/start`

- Creates your wallet (if first time)
- Shows the onboarding app once
- After first /start it is triggering the main menu

<img
  src="/images/prediction-bot/start-menu.png"
  alt="Bot commands menu"
  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}
/>

### `/browse`

Opens the browsing interface to explore events by category:

<img
  src="/images/prediction-bot/browsing.png"

  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}

### `/search`

Search for markets by keyword. Type `/search` and then enter your search term:

<img
  src="/images/prediction-bot/search_command.png"
  alt="Search command showing results for tennis"
  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}
/>

Results show matching events with volume, liquidity, and dates.

### `/leaderboard`

View the top traders leaderboard. Filter by PnL, Volume, or Win Rate across different time periods.

<img
  src="/images/prediction-bot/leaderboard.png"
  alt="Clan leaderboard card showing top 10 traders"
  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}
/>

### `/withdraw`

1. **🏠 Main menu**
2. **💸 Withdraw**
3. Enter the recipient Solana address

<img
  src="/images/prediction-bot/withdraw-input-address.png"
  alt="Withdraw input address screen"
  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}
/>

4. Select token (SOL or USDC)
5. Choose amount (preset percentages or custom)

<img
  src="/images/prediction-bot/withdraw-input-amount.png"
  alt="Withdraw input amount screen"
  style={{ display: 'block', margin: '0 auto', maxWidth: '100%', borderRadius: '0.5rem' }}
/>

6. Confirm the withdrawal
